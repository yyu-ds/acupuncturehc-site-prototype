# Acupuncture Health Center Redesign

Static GitHub Pages site for Acupuncture Health Center.

## Files

- `index.html` is the site entry point for GitHub Pages.
- `old-site-assets/` contains clinic photos and service imagery.
- `service-icons/` contains the cleaned service icons used on the page.
- `he-calligraphy_art.png` is used in the hero artwork.

## Local Preview

From this folder:

```sh
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

## GitHub Pages

After pushing this folder to GitHub, enable Pages in the repository settings:

1. Go to `Settings` > `Pages`.
2. Set source to `Deploy from a branch`.
3. Select branch `main` and folder `/`.
4. Save.
